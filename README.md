## Segundo Envio: Notas de Atualização

Principais alterações realizadas na versão final do **Trabalho 01**:

* **Localização e Rodapés:** Atualização dos endereços em todas as páginas para destacar as duas unidades de Salvador (**CIMATEC** e **Dendezeiros**).
* **Mapa Funcional:** Inserção de um link real do Google Maps no `contato.html`.
* **Validação de Formulário (HTML5):** O campo de telefone agora utiliza os atributos `pattern` e `maxlength` para aceitar exclusivamente números.(Ainda é possível digitar outros caracteres porém surgirá um aviso solicitando a correção dos dados).
* **Página de Sucesso:** Criação do arquivo `sucesso.html` para redirecionamento após o envio correto do formulário.
